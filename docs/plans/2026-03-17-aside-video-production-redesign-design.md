**实现逻辑：**
```typescript
import pinyin from 'pinyin';

function filterRegions(regions: Region[], query: string): Region[] {
  if (!query) return regions;

  const lowerQuery = query.toLowerCase();

  return regions.filter(region => {
    // 中文匹配
    if (region.name.includes(query)) return true;

    // 拼音匹配
    const py = pinyin(region.name, { style: pinyin.STYLE_NORMAL })
      .flat()
      .join('');

    return py.toLowerCase().includes(lowerQuery);
  });
}
```

### 关键功能

#### 1. 左右分栏布局

- **左侧**：Agent 群聊式工作流（40% 宽度）
- **右侧**：可视化画板（60% 宽度）

#### 2. 群聊交互（左侧）

每个 Agent 步骤显示为消息气泡，需要用户确认的操作显示为按钮。

**Agent 工作流步骤：**

1. **🎬 艺术总监 Agent**
   - 确认视频长度（长视频/短视频）
   - 确认视频方向（横版/竖版）

2. **🎭 选角导演 Agent（角色 Agent）**
   - 上传人物参考图 或 自由发挥
   - 生成人物卡片和关系图

3. **🎨 分镜师 Agent**
   - 生成关键帧分镜图
   - 一张大图包含 N×M 个分镜小图

4. **📹 摄像导演 Agent**
   - 根据分镜图生成分镜视频
   - 每个分镜生成对应视频片段

5. **合成**
   - 将所有分镜视频合成成片

#### 3. 可视化画板（右侧）

类似脑图的节点编辑器，支持：

- **人物关系图**：节点 + 箭头连接
- **人物卡片**：显示角色设定、参考图
- **分镜图**：N×M 网格展示所有分镜
- **卡片内按钮**：编辑、重新生成

**数据结构：**
```typescript
// 节点数据
interface DirectorNode {
  id: string;
  type: 'character' | 'scene' | 'storyboard';
  position: { x: number; y: number };
  data: {
    name: string;
    description?: string;
    imageUrl?: string;
  };
}

// 边（关系）数据
interface DirectorEdge {
  id: string;
  source: string;
  target: string;

  gridLayout: {
    rows: number;          // N 行
    cols: number;          // M 列
  };
  scenes: Scene[];         // 分镜场景描述
}
```

#### 5. 分镜图优化

**为什么使用一张大图？**

- **更快速**：一次生成所有分镜，减少网络请求
- **更省 token**：AI 只需一次理解和生成，而非多次
- **更好的一致性**：所有分镜在同一上下文中生成，风格更统一

## 设计原则

1. **渐进式引导**：一步一步引导用户完成，降低认知负担
2. **视觉层次清晰**：主要操作在主要区域，次要信息在次要区域
3. **即时反馈**：操作后立即显示结果（搜索、生成）
4. **灵活性和控制**：导演模式提供精细控制，快速合成提供便捷操作
5. **资源优化**：分镜图使用一张大图，减少网络请求和 token 消耗

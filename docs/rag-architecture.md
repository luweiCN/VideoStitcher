## 4. 相似度检索流程

```
用户查询
  │
  ├─ query: "美食教学视频开场"
  └─ topK: 5
  │
  ▼
┌──────────────────────────┐
│  KnowledgeBase Service   │
│  searchSimilar()         │
└────────┬─────────────────┘
         │
         ▼
┌──────────────────────────┐
│  验证查询文本            │
│  • 不为空                │
│  • 长度合理              │
└────────┬─────────────────┘
         │
         ▼
┌──────────────────────────┐
│  调用豆包 Embeddings     │
│  embedQuery(query)       │
│  → 1536 维查询向量       │
└────────┬─────────────────┘
         │
         ▼
┌──────────────────────────┐
│  VectorStore             │
│  similaritySearch()      │
└────────┬─────────────────┘
         │
         ▼
┌──────────────────────────────────┐
│  SQLite + sqlite-vec 查询        │
│  SELECT                           │
│    v.id,                          │
│    d.content,                     │
│    d.metadata,                    │
│    vec_distance_cosine(           │
│      v.embedding, ?               │
│    ) as distance                  │
│  FROM knowledge_vectors v         │
│  JOIN knowledge_documents d       │
│    ON v.id = d.id                 │
│  ORDER BY distance ASC            │
│  LIMIT ?                          │
└────────┬─────────────────────────┘
         │
         ▼
┌──────────────────────────┐
│  处理查询结果            │
│  for each row:           │
│  • distance → score      │
│  • score = 1 - distance  │
│  • 解析 metadata JSON    │
└────────┬─────────────────┘
         │
         ▼
┌──────────────────────────┐
│  返回结果列表            │
│  [                       │
│    {                     │
│      materialId,         │
│      content,            │
│      score (0-1),        │
│      metadata,           │
│      chunkIndex          │
│    },                    │
│    ...                   │
│  ]                       │
└──────────────────────────┘
```

## 6. 技术栈

| 层次 | 技术 | 用途 |
|------|------|------|
| 用户界面 | React + TypeScript | 渲染进程 UI |
| IPC 通信 | Electron IPC | 主进程与渲染进程通信 |
| 服务层 | TypeScript | 业务逻辑处理 |
| 向量存储 | SQLite + sqlite-vec | 文档和向量存储 |
| 向量生成 | 豆包 Embeddings API | 1536 维文本嵌入 |
| 文本生成 | 豆包 LLM API | 脚本内容生成 |
| 工作流 | LangGraph | 脚本生成流程编排 |

## 7. 性能指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 向量维度 | 1536 | 豆包 Embeddings 输出 |
| 分块大小 | 500 字符 | 平衡检索精度和速度 |
| 分块重叠 | 50 字符 | 避免语义截断 |
| 检索 Top-K | 5 | 返回最相关的 5 个案例 |
| 相似度阈值 | 0.7+ | 建议只使用高相似度结果 |
| 嵌入延迟 | < 100ms | 单次嵌入 API 调用 |
| 检索延迟 | < 50ms | 向量检索（不含嵌入） |
| 总检索延迟 | < 200ms | 完整检索流程 |

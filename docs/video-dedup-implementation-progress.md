# 视频降重处理实现检查点

更新时间：2026-07-13

## 当前状态

视频降重处理已完成首版完整链路：本地变体元素库 → 绿幕配方 → 单轨排程 → 任务中心 → FFmpeg 合成 → 输出文件。

## 已完成

- 新增任务类型 `video_dedup`，任务中心可创建、排队、显示进度、取消、失败重试和记录输出。
- 递归扫描本地元素库，支持 PNG、WebP、GIF、MP4、MOV、M4V。
- 元素库支持分类统计、搜索、筛选、缩略图和异常文件统计。
- 绿幕视频支持独立保存基准色、相似度、边缘柔化和绿色溢出抑制。
- 绿幕参数修改后可以生成真实单帧抠色预览。
- 批量处理页支持生成第一条视频的真实合成预览，并展示实际事件时间轴；参数变化后会提示重新生成。
- 均匀分布和随机分布均使用可复现随机种子。
- 单轨防重叠为引擎硬约束，并校验片头、片尾和元素最小间隔。
- PNG、GIF、绿幕视频可以在同一条输出视频中按不同事件合成。
- GIF 和绿幕短视频支持循环覆盖完整事件持续时间。
- 输出使用 H.264、AAC、CRF 18、`faststart`，保留原视频时长和音频。
- 安全输出采用任务临时目录和唯一文件名，避免并发覆盖现有文件。
- 批量处理与变体元素库只切换页面内部状态，不写入浏览历史；反复切换后左上角一次即可返回首页。
- 修复元素库扫描反复触发和失败缩略图无限重试的问题，避免持续调用 FFmpeg 导致页面卡死。
- 文件选择器补充标准文件输入回退路径，文件对话框、拖放和自动化输入共用同一套校验逻辑。
- 元素库只有一个绿幕视频时会自动选中该元素并生成右侧抠色预览；批量预览生成后自动定位到首个元素事件的中间时刻。
- 数据库连接改为完整初始化成功后才发布，并增加 5 秒忙等待，避免初始化失败后继续使用半失效连接。
- 主进程增加单实例锁，阻止开发重载残留进程重复占用同一任务数据库。
- 批量处理页新增全局“元素尺寸”，支持小 12%、中 22%、大 32% 和 5%～50% 自定义滑杆；尺寸变化会使旧预览失效，并同步写入正式任务配置。
- 元素库分类固定按扩展名执行，并将“全部”视图改为三个独立分组，避免平铺换列造成素材归类错觉。
- 真实预览时间轴支持拖动事件块调整出现位置、拖动左右边缘调整持续时长；编辑后可使用原事件表重新渲染。
- 手动时间轴不受自动生成规则的最短/最长持续限制；仅保留 0.1 秒下限、片头片尾、最小间隔和单轨防重叠硬约束，并可写入首条视频的第 1 个正式变体任务。
- 时间轴将静态贴纸、GIF 动态贴纸、绿幕视频固定显示为蓝色、紫色和绿色，颜色不受全局皮肤覆盖。
- 时间轴支持从完整元素库随机增加一个事件，并在单轨约束下自动寻找可用空档；支持选中红色事件后通过工具栏删除。

## 关键实现文件

- `src/shared/videoDedup/types.ts`：共享配置与事件类型。
- `src/shared/videoDedup/scheduler.ts`：均匀/随机排程与单轨校验。
- `src/main/ipc/videoDedup.ts`：元素库扫描、绿幕配方和抠色预览。
- `src/main/services/VideoDedupEngine.ts`：FFmpeg 合成引擎。
- `src/main/services/TaskQueueManager.ts`：任务中心执行接入。
- `src/renderer/features/VideoDedupMode.tsx`：完整操作界面。
- `scripts/test-video-dedup-engine.ts`：独立端到端引擎测试。
- `scripts/test-video-dedup-ui.cjs`：隔离用户数据的真实 Electron 入口与任务链路验收。
- `scripts/test-video-dedup-scale-ui.cjs`：元素尺寸默认值、选中态、自定义滑杆和持久化验收。
- `scripts/test-video-dedup-timeline-ui.cjs`：时间轴拖动、边缘拉伸、三类颜色与编辑后重新渲染验收。

## 已验证

- 生产构建：`npm run build` 通过。
- 元素库：静态图、GIF、绿幕视频扫描通过。
- 排程器：均匀分布与随机分布均通过单轨防重叠校验。
- 绿幕预览：透明 PNG 预览生成成功。
- 实际合成：8 秒、640×360 测试视频成功叠加 PNG、GIF 和绿幕 MP4。
- 输出检查：视频为 H.264，音频为 AAC，时长保持 8 秒。
- 画面检查：PNG 位于左上、GIF 位于右上、绿幕抠色结果位于右下，三个事件互不重叠。
- 真实 Electron UI：从首页进入、切换变体元素库、导入视频、生成真实预览、添加任务全部通过。
- 任务中心：视频降重任务状态为“已完成”，输出目录生成可播放 MP4 成品。
- 返回逻辑：连续切换批量处理/变体元素库 4 轮后，左上角一次返回首页。
- 截图自检：已保存首页入口、批量处理、元素库、真实预览、任务弹窗、任务中心完成记录和一次返回首页共 7 张截图。
- 单绿幕专项验收：元素库仅包含一个绿幕 MP4 时，抠色预览、批量真实预览、任务中心导出均通过；成品抽帧确认主体保留且绿色背景已移除。
- 真实业务素材验收：使用 `小猫.mp4` 与实际 15 秒原视频完成预览和任务导出；旧默认参数 42%/18% 会使整帧 Alpha 归零，现已调整为相似度 12%、边缘柔化 5%、溢色抑制 20%。
- 旧默认绿幕配方会在读取时迁移到安全参数；若用户调参导致主体被完整扣空，预览和任务创建会给出明确错误并阻止静默导出。
- 正式任务库副本通过 `integrity_check` 与 `quick_check`，数据文件未损坏；`disk I/O error` 来源于三个并存主进程和失效连接，而非绿幕计算或磁盘容量。
- 元素尺寸专项验收：默认 22% 与自定义 41% UI 状态均通过截图检查；41% 已完成真实预览和正式任务导出，任务配置记录为 `0.41`。
- 时间轴专项验收：事件整体拖动、右边缘拉伸、待重新渲染状态和编辑后重渲染均通过；重渲染后开始与结束时间保持不变。
- 类型配色专项验收：静态贴纸、GIF 动态贴纸、绿幕视频的计算色值分别为蓝色 `#0EA5E9`、紫色 `#8B5CF6`、绿色 `#10B981`。
- 新增删除专项验收：随机新增后事件数从 1 增加到 2，新事件与删除按钮均为红色选中态，原事件保持类型色；删除后事件数恢复为 1，单轨最小间隔校验通过。

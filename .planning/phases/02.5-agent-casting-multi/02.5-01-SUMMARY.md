### 文件结构

```
src/shared/constants/castingDirectorMultiTemplates.ts
├── CASTING_PLANNER_AGENT_EDITABLE_PART      # 选角规划师人设、角色分析方法论
├── CASTING_PLANNER_AGENT_LOCKED_PART        # 角色视觉规格 JSON 格式定义
├── CASTING_PLANNER_AGENT_USER_PROMPT_TEMPLATE  # 变量：artDirectorOutput, scriptContent
├── CASTING_PLANNER_AGENT_BUILTIN_TEMPLATE   # Planner 元数据
├── CASTING_VISUALIZER_AGENT_EDITABLE_PART   # 形象可视化师人设、提示词构建方法论
├── CASTING_VISUALIZER_AGENT_LOCKED_PART     # 图像生成提示词模板
├── CASTING_VISUALIZER_AGENT_USER_PROMPT_TEMPLATE  # 变量：characterVisualSpecs, styleGuide
└── CASTING_VISUALIZER_AGENT_BUILTIN_TEMPLATE # Visualizer 元数据
```

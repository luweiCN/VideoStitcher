# Phase 02.5 Plan 02: 选角导演多阶段 Agent 实现

## 执行摘要

创建选角导演多阶段 Agent 实现，包含 Planner 和 Visualizer 两个阶段的独立函数。

## 实现详情

### 1. 类型定义

定义了以下类型接口：
- `CastingPlannerInput` / `CastingPlannerOutput` - Planner 阶段输入输出
- `CastingVisualizerInput` / `CastingVisualizerOutput` - Visualizer 阶段输入输出
- `CharacterVisualSpec` - 角色视觉规格
- `StyleGuide` - 风格指南

### 2. runCastingPlannerAgent 函数

职责：根据艺术总监输出和剧本，生成角色视觉规格 JSON

- 加载 Planner 提示词模板（`CASTING_PLANNER_AGENT_*`）
- 调用 LLM 生成角色视觉规格
- 解析 JSON 输出并验证
- 返回 `CastingPlannerOutput`

### 3. runCastingVisualizerAgent 函数

职责：根据角色视觉规格，生成角色参考图

- 加载 Visualizer 提示词模板（`CASTING_VISUALIZER_AGENT_*`）
- 构建图像生成提示词
- 调用图像生成 API
- 返回 `CastingVisualizerOutput`（含图像 URL）

### 4. runCastingDirectorAgent 函数更新

新增功能：
- `options.useMultiStage?: boolean`（默认 false，保持兼容）
- 当 `useMultiStage=true` 时：先调用 Planner，再调用 Visualizer
- 当 `useMultiStage=false` 时：保持现有单阶段逻辑

### 5. 导演模式支持

导演模式在 Node 层处理（`casting-director.ts`），与 storyboard-artist 模式一致：
- 当 `state.executionMode === 'director'` 时设置 `humanApproval: false`
